How to use
1
Start the MCP server: cd packages/mcp-server && npm start
2
Open your dev site on localhost and click Annotate Current Page
3
Click any element on the page to drop a numbered pin and add a comment
4
Click Send to Claude in the annotation toolbar
5
In Claude Code, type: Check my UI feedback
Only works on localhost
